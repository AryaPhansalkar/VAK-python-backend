# V.A.K API

This project is a simple **Flask-based API** that detects abusive or toxic comments using a pre-trained machine learning model and a TF-IDF vectorizer.  

---

## Features

- Detects abusive comments in real time  
- RESTful API built using **Flask**  
- Supports **CORS** for frontend integration  
- Uses a **TF-IDF Vectorizer** and trained ML model stored as `.pkl` files  

---
## Project Setup

First step is setting up the development environment and installing all the required dependencies
with:

``` shell
python -m venv venv
# On Windows
venv\Scripts\activate 
.\make.cmd Setup   
```
This will setup all the requirements for the project in the virtual environment.
